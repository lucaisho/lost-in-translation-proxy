Lost in Translation — Privacy Policy

Last updated: April 2026

Overview

Lost in Translation is a Chrome extension that audits in-language web pages for English bleed, translation issues, and meta tag gaps. This privacy policy explains what data the extension accesses, how it is used, and what is not collected or stored.

Data accessed during an audit

When you run an audit, the extension accesses the following from the active browser tab:

  • The visible text content of the page being audited
  • The page URL
  • HTML meta tags (title, meta description, Open Graph tags, canonical tag)
  • Same-domain hyperlinks present on the page

For in-language pages, the extension also silently loads the equivalent English (EN-US) version of the same URL in a background tab to compare meta tag values.

How data is used

The page text and meta tag content collected above is sent to a secure serverless proxy hosted on Vercel, which forwards the request to the Google Gemini API for linguistic analysis. The analysis identifies potential translation issues and meta tag gaps and returns findings to the extension for display and export.

Data is transmitted solely for the purpose of generating audit findings. It is not used for any other purpose.

Data retention

The extension does not store, log, or retain any page content, URLs, or analysis results. Data sent to the Vercel proxy is processed in real time and not persisted. No database or external storage is used.

What is not collected

  • No personally identifiable information (PII)
  • No login credentials or authentication tokens
  • No browsing history
  • No data from pages the extension is not actively auditing
  • No financial or payment information

Third-party services

The extension uses the following third-party services:

  • Google Gemini API — page text is sent to Gemini for linguistic analysis. Google's privacy policy: https://policies.google.com/privacy
  • Vercel — the serverless proxy is hosted on Vercel. Vercel's privacy policy: https://vercel.com/legal/privacy-policy
